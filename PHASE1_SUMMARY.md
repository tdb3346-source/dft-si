# Phase 1 findings - SrTiO3, the clean demonstration

- Converged settings: PW 900 eV (highest of any material so far - oxygen's compact 2p orbitals need fine plane-wave detail), k-grid 6x6x6.
- PBE gap: 2.110 eV vs experimental 3.20 eV -> -34%% underestimation. Textbook PBE failure, correct direction, expected magnitude.
- Reference values (lattice 3.905 A, gap ~3.2-3.25 eV) verified against 5 independent literature sources.
- Combined with the CsPbI3 result: PBE's gap error is NOT one fixed number. It can be REVERSED by missing spin-orbit coupling in heavy-element materials.
